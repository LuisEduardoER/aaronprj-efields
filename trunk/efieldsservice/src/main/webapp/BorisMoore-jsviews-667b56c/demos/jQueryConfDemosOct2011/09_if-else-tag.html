13 Using {{if}} and {{else}} to render conditional sections

TitleLanguages

HTML:

<td>
	{{#if Languages}}
		...
	{{else Subtitles}}
		....
	{{else}}
		...
	{{/if}}
</td>

Data:

var movies = [
	{
		Title: "Meet Joe Black",
		Languages: "English and French",
		Subtitles: "English"
	},
	{
		Title: "Eyes Wide Shut",
		Subtitles: "French and Spanish"
	},
	...
];
