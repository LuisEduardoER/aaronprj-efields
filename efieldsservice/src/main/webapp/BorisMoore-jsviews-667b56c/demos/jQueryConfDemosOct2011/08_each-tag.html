11 Using {{#each}} to render hierarchical data - inline nested template

TitleLanguages

Data:

var movie = {
	Title: "Eyes Wide Shut",
	Languages: [
		{ Name: "French" },
		{ Name: "Mandarin" },
		{ Name: "Spanish" }
	]
}

HTML:

<td>{{=Title}}</td>
<td>
	{{#each Languages}}
		<div>{{=Name}}</div>
	{{/each}}
</td>
